Story App adalah aplikasi android seperti instagram yaitu dengan berbagi cerita kepada pengguna lain

## Features
* Maps
* Upload Story
* Share Location 

## Technology Used
* Retrofit
* MVVM 
* Room DB
* Maps Api
* Pagination (Paging 3)
* Dependency injection (Dagger Hilt)
* Unit Testing
* Custom View
* Camera
